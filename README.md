# Custom Quiz Interface Engine

A sleek, responsive web interface built for UI/UX testing and custom exam simulation practice.

## Features
* 100% locally hosted.
* Completely stateless.
* Resets memory on browser refresh.
* Zero data tracking.

## How to Run
1. Clone or download this repository.
2. Open `index.html` directly in your browser.

## Built With
This project was "proudly", though entirely developed using:
* Claude-fable-5
* Gemini-3.1-pro-preview
* Gemini-3.1-Pro
