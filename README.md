# PBQ Simulator

A sleek, responsive [web](https://vosdyh.github.io/ITcore2sim/) interface built for UI/UX testing and custom exam simulation practice. The engine is currently in a stable, alpha-ready state.

## Features
* 100% locally hosted.
* Retain score history across browser refreshes.
* Generates dynamic, CompTIA-style, adaptive SVG performance graphs
* No external telemetry or data collection.

## Usage & Installation

This application is a single-file static tool, making deployment and usage straightforward:

1. **Direct Access (No Installation)**: You can access and run the simulator directly in your browser without downloading any files by visiting the project’s [GitHub page](https://vosdyh.github.io/ITcore2sim/).
2. **Local Execution**: Alternatively, you can clone or download this repository to your local machine and open the `index.html` file directly in any modern desktop web browser (Chrome, Firefox, Edge, Safari).
3. **Development Setup**: For the best development experience when modifying the interface, serve the file using a local web server, such as the "Live Server" extension in VS Code.
