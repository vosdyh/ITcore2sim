# Custom Quiz Interface Engine

A sleek, responsive [web](https://vosdyh.github.io/ITcore2sim/) interface built for UI/UX testing and custom exam simulation practice.

## Features
* 100% locally hosted.
* Retain score history and attempt progress across browser refreshes.
* Generates dynamic, CompTIA-style, adaptive SVG performance graphs 
* No external telemetry or data collection.

## How to Run
1. Clone or download this repository.
2. Open `index.html` directly in your browser.
