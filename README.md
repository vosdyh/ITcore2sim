# PBQ Simulator

## Project Overview
---
This project is a static, single-file PBQ (Performance-Based Question) simulator built for educational practice. The engine is currently in a stable, alpha-ready state, providing a reliable environment for custom exam simulation.

## Core Features
---
* **Portable Architecture:** A zero-dependency, single-file static web application that runs entirely offline.
* **Interactive Simulator Engine:** A browser-based engine that handles user input, command-line interactions, and status updates without a backend.
* **Persistence Layer:** Integrated local storage functionality to track performance history and user progress across sessions.
* **Responsive UI:** A dynamic, flexbox-based layout that adapts between a full-width workspace and a minimized navigation view.
* **Anti-Meta-Gaming Logic:** Built-in randomization to ensure parameter sets change with each use, requiring live analysis rather than rote memorization.

## Usage
---
Open `index.html` in any modern web browser.
